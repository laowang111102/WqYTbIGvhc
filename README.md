# Python图书馆管理系统 python1012

## 项目概述

本项目是一个基于Python语言的图书馆管理系统，旨在为图书馆提供高效便捷的管理服务。

## 技术栈

- 后端：Django
- 数据库：MySQL
- 前端：Vue.js

## 功能模块

### 核心功能

- 图书信息管理
- 图书馆信息管理
- 留言板
- 公告信息管理
- 个人中心

### 管理模块

- 借阅信息管理
- 归还信息管理
- 图书类型管理
- 图书管理
- 罚金信息管理
- 留言板管理

## 系统角色

- 系统管理员
- 图书管理员
- 普通用户

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x
- Vue.js 2.x
